# bash-demo

This is a basic demo and list of common bash knowledge you may be expected to know.

## Contents

Feel free to jump around, but a reasonable order of progression might be:

* [shebang](./shebang/README.md): `#!/bin/bash`
* [set](./set/README.md): `set -Eeuo pipefail`
* [expansion](./expansion/README.md): `echo ${FOO:-BAR}`
* [parameters](./parameters/README.md): `$0`, `$1`, `$@` ... etc
